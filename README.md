# Dyanmic_Budget_Tracker
A dynamic personal budget tracker built in Excel that monitors monthly income, expenses, and savings in real time. Navigation arrows move between months, automatically updating all figures — income breakdown, expense categories, month-end balance, and year-over-year change. A Top 5 Expenses donut chart and dual income/expense trend charts provide a full-year view at a glance. Designed for anyone who wants clarity on where their money goes without leaving Excel.
